GOV.UK Content API v0.1.0

Scroll down for example requests and responses.

An API for accessing the content available on GOV.UK.

Base URLs:

  • https://gov.uk/api/content

Default

GET /{id}

Get a piece of content by ID.

Returns a single piece of content for an ID.

Parameters

Parameter In Type Description
id path integer ID of the thing to return. (required)

Example responses

Responses

Status Meaning Description Schema
200 OK Successful operation. Content
404 Not Found No content with the ID specified. None
This operation does not require authentication

GET /lookup/{base_path}

Get a piece of content by base path.

Returns a single piece of content for a base path.

Parameters

Parameter In Type Description
base_path path string Path of the piece of content to return. (required)
date query string(date-time) Date at which to access the conent, defaults to right now.

Example responses

Responses

Status Meaning Description Schema
200 OK Successful operation. Content
404 Not Found No content at the path specified. None
This operation does not require authentication
